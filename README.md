# 致乖乖 - 爱的粒子特效 ❤️

这是一个为庄依（乖乖）特别制作的交互式爱心烟花特效网页。

## 功能特色 ✨

- 🎆 **多种烟花效果**：心形烟花、彩虹烟花、螺旋烟花等
- 💖 **交互式体验**：点击屏幕任意位置生成烟花
- 📱 **移动端优化**：完美适配手机和平板
- 🎵 **背景音乐**：浪漫钢琴曲配乐
- 🎨 **粒子特效**：使用particles.js库创建精美背景

## 如何使用 🚀

1. **在电脑上**：
   - 点击按钮触发不同烟花效果
   - 点击屏幕任意位置生成小烟花
   - 使用音乐控制按钮播放/暂停背景音乐

2. **在手机上**：
   - 点击按钮观看特效
   - 点击屏幕生成烟花
   - **双击**屏幕生成爱心烟花
   - **长按**屏幕生成螺旋烟花
